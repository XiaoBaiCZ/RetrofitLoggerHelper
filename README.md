### RetrofitLoggerHelper

> Retrofit / OkHttp 使用的 Logger 助手

##### 添加方式
~~~
allprojects {
  repositories {
    ...
    maven { url 'https://www.jitpack.io' }
  }
}
~~~
~~~
dependencies {
  implementation 'com.github.XiaoBaiCZ:RetrofitLoggerHelper:0.1'
}
~~~
